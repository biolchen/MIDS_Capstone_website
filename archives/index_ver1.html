    OUR GOAL

Problem Statement and Our Goal

Contrastive learning is a big deal for all these reasons: 1. There is lots of unlabeled data; 2. It is expensive to label; 3. This problem is just going to get bigger with more sensors The most popular algorithm for constrastive learning is cycleGAN. However, a new model, named C.U.T., was reported to be faster and more efficient. We would like to compare these two algorithm.

Our main goal is to Test CUT to see if it outperforms cyclegan at domain adaptation in satellites images. We will test the effectiveness of Contrastive Learning for Unpaired Image-to-Image Translation (C.U.T.) when used for domain adaptation in satellite imagery. Our approach is to apply domain adaptation, converting images taken in the summer into synthetic spring, fall, and winter images. We will compare the results of a model trained with synthetic images generated with C.U.T with the results of a model trained with images generated with CycleGAN.

Our experimental plan

Step1:Baseline Validation Accuracy

 We begin with a labeled training set consisting of summer images and a labeled validation set consisting of winter images. We train a model using the summer images and then try to predict winter images with this model. This results in a baseline evaluation accuracy.

Results

Step2:Generate Fake Winter Images

We then train the CUT model with the labeled summer images and randomly selected winter images. This results in synthetic versions of our summer images that have been “winterized”, but retain the original labels. This creates a batch of fake labeled winter images.

Examples

Step3:Retrain Model and Compare Results

We then retrain our classification model with the original labeled summer images combined with the new fake labeled winter images. The model is then used to predict the validation winter images, resulting in a new validation score. If there is an improvement in validation accuracy, it is a result of the domain adaptation through the CUT model.

Our Methods

Domain Adaptation

Uses one labelled dataset from one domain (the source) to perform classification on a dataset on a new domain (the target)

CycleGAN

  • Provided framework for unpaired image-to-image translation
  • Very popular choice for domain adaptation
  • Trains two-sided

    C.U.T.

  • Image to Image Translation
  • Generator (Encoder/Decoder) and Discriminator
  • Patchwise Contrastive Learning
  • Trains One-sided, making it 2x faster than CycleGAN
